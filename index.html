Bagian 1 — Persiapan Sebelum Publish
1
Buka Google Sites & Login
🌐
  • a
    Buka browser (Chrome lebih disarankan) dan pergi ke sites.google.com
  • b
    Login menggunakan akun Google sekolah kamu
  • c
    Klik nama proyek website kamu untuk membukanya
sites.google.com
Google Sites
2
Cek Nama Website (Judul & Title)
✏️
  • a
    Pastikan nama website di pojok kiri atas sudah diisi (bukan "Untitled Site")
  • b
    Klik nama website untuk mengeditnya jika perlu
  • c
    Gunakan nama yang jelas dan relevan dengan topik website kamu
💡Contoh nama yang baik: "Blog Kesehatan Remaja", "Panduan Belajar Matematika", "Komunitas Pecinta Alam Kelas 7C"
3
Periksa Halaman-halaman Website
📄
  • a
    Klik ikon Pages (ikon halaman bertumpuk) di panel kanan untuk melihat daftar halaman
  • b
    Pastikan ada minimal 3 halaman: Home, Isi/Konten, dan Tentang atau Kontak
  • c
    Klik setiap halaman dan cek isinya — pastikan tidak ada teks kosong atau placeholder

Bagian 2 — Proses Publish
4
Klik Tombol "Publish" Langkah Utama!
🚀
  • a
    Cari tombol "Publish" berwarna biru di pojok kanan atas halaman editor
  • b
    Klik tombol tersebut — akan muncul jendela dialog pengaturan publish
sites.google.com/u/0/d/abc123.../edit
Sites
← Klik tombol "Publish" yang bersinar ini!
5
Atur Web Address (Alamat Website)
🔗
  • a
    Setelah klik Publish, akan muncul jendela "Publish to the web"
  • b
    Di bagian "Web address", ketik nama URL yang kamu inginkan (hanya huruf kecil, angka, dan tanda hubung)
  • c
    Contoh: kalau nama kamu Budi, bisa tulis website-budi-7c
  • d
    Pastikan muncul tanda ✅ yang artinya nama URL tersedia

6
Klik "Publish" di Dialog & Konfirmasi
✅
  • a
    Klik tombol "Publish" biru di dalam jendela dialog tersebut
  • b
    Jika ada pertanyaan tentang siapa yang bisa melihat website, pilih "Anyone with the link" atau "Public"
  • c
    Klik OK atau Confirm jika muncul konfirmasi tambahan
  • d
    Tunggu beberapa detik hingga proses selesai

Bagian 3 — Salin & Bagikan Link
7
Salin Link Website Kamu
📋
  • a
    Setelah publish berhasil, klik tombol "Copy link" atau cari ikon berbagi (⋮) → "Copy link"
  • b
    Atau buka tab baru dan pergi ke URL: sites.google.com/view/[nama-url-kamu]
  • c
    Salin URL dari address bar browser
sites.google.com/view/website-budi-7c
💡Test dulu linknya — buka tab baru (Ctrl+T), paste linknya, dan pastikan website bisa dibuka dengan benar!
8
Kumpulkan Link ke Guru
📤
  • a
    Tulis nama kamu dan link website di papan tulis atau kirim melalui form yang diberikan guru
  • b
    Pastikan link yang dikumpulkan bisa dibuka oleh orang lain, bukan hanya kamu sendiri
  • c
    Format pengumpulan: Nama — Link Website
Contoh Pengumpulan
Budi Santoso — sites.google.com/view/website-budi-7c
Siti Rahayu — sites.google.com/view/siti-blog-7c
Masalah Umum & Solusinya
❌ Tombol Publish tidak aktif
Kemungkinan ada error di halaman. Coba refresh browser dan cek apakah semua halaman sudah tersimpan (auto-save).
❌ URL sudah dipakai orang lain
Ganti nama URL dengan menambahkan angka atau huruf unik. Contoh: website-budi-7c-2026
❌ Link tidak bisa dibuka teman
Pastikan pengaturan akses sudah "Anyone with the link" atau "Public". Buka ⋮ → Settings → Visibility.
❌ Perubahan tidak muncul setelah publish
Setiap kali ada perubahan, kamu harus klik Publish lagi. Perubahan otomatis tersimpan tapi tidak otomatis terpublish.
